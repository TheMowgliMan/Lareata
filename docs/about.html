About Lareata

Lareata is a sandbox-type game in the vein of Terraria, created by Aspect Inc. in 2021. Aspect Inc. is not a legal entity, it is an alias. It is developed in Godot and is planned to be relesed for Linux, Mac, and Windows. It is licensed undel the GPL V3 license and thus is able to be redistributed and modified for free. However, we may go commercial in the future.

We would greatly appreciate it if you would help develop it! Click the "Contribute" link in the navbar to learn how.
