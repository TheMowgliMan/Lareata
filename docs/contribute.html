How to Contribute

If you want to contribute to Lareata, feel free to test our early versions... For free!
We currently only have Windows 64-bit versions, but we'll get to the Mac and Linux versions soon. When your done testing (Or while testing!), let me know what you think at the Lareata Discussions. Pick the topic for the version released! Please note that your browser may think this file is a threat, please ignore them and click "keep anyway" even though the browser dislikes the file type. You may also need to let it through your antivirus program.
